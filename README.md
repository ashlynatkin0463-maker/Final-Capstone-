# CAPSTONE-
Apartments for Rent Analysis in the United States
Capstone Project | Rental Housing Market Insights

By: Ashlyn Atkin

Project Overview

The U.S. rental housing market is shaped by geographic location, property characteristics, and listing features such as size, amenities, and segmentation. These factors collectively influence pricing patterns across different regions.

This project analyzes approximately 99,000 rental listings across the United States to identify key drivers of rental prices and uncover geographic and market-level trends. Using interactive Power BI dashboards, the analysis explores how property attributes and location impact affordability and rental demand.

Dataset Description

The dataset contains ~99,823 rental listings with 20 variables describing property details, pricing, and geographic information.

Key Variables:
Price – Monthly rental cost
Bedrooms / Bathrooms – Property structure
Square Feet – Property size
State / City – Geographic location
Amenities – Included features
Pets Allowed – Pet policy
Has Photo – Listing visibility indicator
Segment – Market classification (Economy, Standard, Premium, Luxury)

Objectives

This project investigates the following:

What factors most influence rental prices?
How do property features (size, bedrooms, bathrooms) affect rent?
How do rental prices vary across U.S. states and cities?
Where are the most expensive and most affordable rental markets?
How do different market segments (Economy → Luxury) compare in pricing?
Do listing features (e.g., photos, amenities) impact rental value?

Methodology

The analysis was conducted using a structured data workflow:

Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Feature engineering and segmentation
Geographic analysis using state and city-level mapping
Statistical aggregation of rental metrics
Interactive dashboard development in Power BI
Visualization of trends, distributions, and correlations

Key Findings
Rental prices show strong geographic variation, with coastal and urban states dominating high-cost markets.
Property size (square footage) is the strongest predictor of rent.
Luxury and Premium segments are significantly concentrated in high-demand metropolitan areas.
Economy listings dominate in lower-cost inland regions.
States such as California, New York, and Hawaii represent the highest rental averages.
States like Arkansas and West Virginia represent the lowest rental averages.
Listings with larger square footage and higher bedroom counts consistently command higher prices.
